# TICTECHTALK
### 팀 소개
* **최정혜 (조장)**
  * SPRING SECURITY를 이용한 로그인(유저 관리), 회원 간 1:1 채팅(백엔드)
* 김은별
  * 게시판 관련 전체 기능, 댓글 관련 전체 기능(프론트엔드, 백엔드)
* 김채연
  * 디자인, 퍼블리싱, 프론트 API 통신 관리, 리덕스 툴킷 관리, 회원 간 1:1 채팅(프론트엔드)
 ### 프로젝트 소개
 IT를 주제로 한 토론, 질문, 자료 게시판과 유저 간 1:1 채팅이 가능한 웹 커뮤니티
